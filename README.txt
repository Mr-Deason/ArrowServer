Agent:
	1. enter "make" in folder Agent/
	2. enter "java AgentServer" to start agent

Server:
	1. enter "make" in folder Server/
	2. enter "java Server [agent address] [agent port]" to start server

Client:
	1. enter "make" in folder Client/
	2. enter "java Client [agent address] [agent port]" to start client


[NOTE]
start order: Agent -> Server -> Client
